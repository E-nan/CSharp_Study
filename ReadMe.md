### Visual Studio 2019
- 기능
  - 클래스의 Property 만들기 (getter, setter)
    - prop 입력 후 Tab 두번
    - propg 는 get은 되지만 private set 상태로 만들어짐
  - 코드 좌측 몇번째 줄인지 나타내는 숫자 오른쪽에 생성되는 초록색/노란색 부분은 각각 수정된 상태, 저장된 상태를 뜻한다. 
  

- 단축키

  - Ctrl + K, Ctrl + D  => 자동 정렬
  - Ctrl + K, Ctrl + C  => 주석
  - Ctrl + K, Ctrl + C  => 해제
